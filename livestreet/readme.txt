по поводу настроек конфигов админки. может быть ситуация когда нужны будут актуальные данные конфига (из бд) в классе плагина (PluginААА) в Init(), но на данный момент их там нет, т.к. загрузка конфигов идет по хуку engine_init_complete, который выполняется сразу после инициализации плагинов.
поэтому предлагаю добавить ещё один хук в Engine.class.php в Init() перед инициализацией плагинов (перед строкой):

  /**
   * Инициализируем загруженные плагины
   */
  $this->InitPlugins();

что-то типа

$this->Hook_Run('engine_base_init_complete');
таким образом данные загрузятся из бд и в конфигах будут актуальные значения даже в момент инициализации плагинов.

в результате код будет такой:

  $this->Hook_Run('engine_base_init_complete');
  /**
   * Инициализируем загруженные плагины
   */
  $this->InitPlugins();
  /**
   * Запускаем хуки для события завершения инициализации Engine
   */
  $this->Hook_Run('engine_init_complete');


и перевесить загрузку в хуке из engine_init_complete на engine_base_init_complete (созданный хук)
