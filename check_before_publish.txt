
Перед публикацией плагина нужно проверить моменты:

1. проверить списки ключей, чтобы не попали ключи, которые редактировать нельзя

2. Проверить обычный вид пхп-массива - сейчас он отключен и код падает если на странице настроек есть такой массив т.к. его значение не отсылается.
  возможно, просто запретить на уровне разработчиков пока использовать такой тип массива.
  сделал пока ексепшн при попытке сохранить такой массив
  
  пример http://ls-admin.com/admin/settings/interface/

3. положить в папки шаблонов синьйо и девелопер их описание (хмл и скриншот)

4. в шаблонах отключить принудительную запись использования темы:
  //$config['view']['theme'] = 'default';                 // 18:13 30.07.2013


5. Заново пересмотреть все наборы настроек (Сделать дифф конфига лс 1.0.3 и актуальной версии лс) и добавить/изменить параметры

6. Настройки конфига админки добавить в интерфейс т.к. сейчас там только тестовые настройки для отладки
